# 数据库

* 事务

  | 隔离等级         | 问题                                                         |
  | ---------------- | ------------------------------------------------------------ |
  | read uncommitted | 脏读：一个事务读到另一个未提交事务写入的数据                 |
  | read committed   | 不可重复读：一个事务读取之前读过的数据，发现已被另一个事务修改 |
  | repeatable read  | 幻读：一个事务插入数据，另一个事务两次读取不一致             |
  | serializable     |                                                              |
