## 设计模式

### 单例模式(Singleton pattern)

```c++
//c++11后标准版
class Singleton {
  private:
    Singleton() = default;
    Singleton(const Singleton&) = delete;
    Singleton& operator=(const Singleton&) = delete;
  public:
    static Singleton& getInstance(){
        static Singleton value;
        return value;
    }
};
```
